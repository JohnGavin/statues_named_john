# Classify subject type

Categorizes subjects into "John", "Woman", "Dog", or "Other".

## Usage

``` r
classify_subject(name, gender, subjects_list = NULL)
```

## Arguments

- name:

  Character name

- gender:

  Character gender ("male", "female")

- subjects_list:

  List or vector of subject tags (e.g., "dog", "animal")

## Value

Character string category
