# Get detailed information about a specific subject

Get detailed information about a specific subject

## Usage

``` r
get_subject(subject_url)
```

## Arguments

- subject_url:

  Character string, URL or slug of the subject

## Value

A list containing detailed subject information including associated
memorials

## Examples

``` r
if (FALSE) { # \dontrun{
subject <- get_subject("/subjects/john-lennon")
} # }
```
