# Get detailed information about a specific memorial

Get detailed information about a specific memorial

## Usage

``` r
get_memorial(memorial_url)
```

## Arguments

- memorial_url:

  Character string, URL or slug of the memorial

## Value

A list containing detailed memorial information

## Examples

``` r
if (FALSE) { # \dontrun{
memorial <- get_memorial("/memorials/john-lennon-statue")
} # }
```
