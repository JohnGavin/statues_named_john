# Search for memorials

Search for memorials

## Usage

``` r
search_memorials(query, pages = 1)
```

## Arguments

- query:

  Character string, search term

- pages:

  Integer, number of pages to fetch (default: 1)

## Value

A tibble containing search results

## Examples

``` r
if (FALSE) { # \dontrun{
results <- search_memorials("John", pages = 3)
} # }
```
